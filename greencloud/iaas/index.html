GREENCLOUD IaaS

Virtuelle Server nach Maß – sicher, flexibel und hochverfügbar

  • Hochverfügbare, leistungsstarke, virtuelle Windows Server
  • Ausgelegt auf Echtzeit-Anwendungen wie Remotedesktop und Telefonie
  • Flexibel anpassbar mit monatlicher Abrechnung
  • Virtuelle Firewalls mit anpassbarem Funktionsumfang
  • Virtuelle Netzwerke für komplexere Strukturen

Einfach

GREENCLOUD IaaS ermöglicht es ihnen, unkompliziert annähernd jedes unter Microsoft Windows betriebene Programm in die Cloud auszulagern – selbst dann wenn Seitens des Herstellers keine Cloud-Variante existiert. Sie erhalten persönliche virtuelle Server und können sie so nutzen, wie Sie es von eigenen Servern gewohnt sind – nur mit weniger Aufwand. Alles, was Sie vor Ort oder für den mobilen Zugriff noch benötigen, ist ein Internetzugang und die gewünschten Endgeräte (PCs, Notebooks, Tablets).
Für Sie entfällt die kostspielige Anschaffung und Unterhaltung eigener Server sowie der zugehörigen Infrastruktur wie Klimatisierung, Stromversorgung und Platzbedarf. Gleichzeitig erhalten Sie eine hohe Stufe der Sicherheit und Verfügbarkeit für Ihre Server, die bei Ihnen vor Ort nur mit hohen Kosten und Aufwendungen realisierbar wäre.

Sicher

Die virtuellen Server werden von uns auf hochverfügbaren Clustern mit gespiegelten Datenspeichern in unserem TÜViT Level 3 zertifizierten Rechenzentrum betrieben. Bei einem Hardware-Defekt sind Ihre virtuellen Server innerhalb von Sekunden auf einer anderen Hardware wieder verfügbar.
Ihre Server werden in einem abgeschotteten Netzwerk und hinter einer persönlichen, frei konfigurierbaren Firewall betrieben, die in verschiedenen Abstufungen an den jeweiligen Sicherheitsbedarf anpassbar ist und über mindestens eine eigene externe IP-Adresse verfügt.
Um auch komplexe Umgebungen realisieren zu können, haben Sie die Möglichkeit beliebig viele zusätzliche Firewalls, IP-Adressen und Netzwerke zu erstellen.

Flexibel

Die virtuellen Server und Firewalls sind frei konfigurierbar, und können Ihren Bedürfnissen jederzeit kurzfristig angepasst werden, beispielsweise wenn Ihre Firma wächst oder sich an saisonale Veränderungen anpassen soll.
Sie bestimmen die Taktung und Anzahl der Prozessorkerne, die Größe des Arbeitsspeichers, sowie die Größe, Geschwindigkeit und Anzahl der Festplatten.
Die Server sind dabei auf anspruchsvollste Echtzeit-Anwendungen wie Remotedesktop und Telefonie ausgelegt. Vom einzelnen Server bis hin zu komplexen Umgebungen ist somit alles möglich, ohne dass Sie selbst in teure Hardware investieren müssen, die eventuell schon nach wenigen Jahren nicht mehr Ihren Anforderungen entspricht.
Zusätzlich kann physische Hardware integriert werden wie z.B. USB-Dongles für Ihre Software, eigene Firewalls und Server im Hosting und weiteres.
