GREENCLOUD IaaS

Virtuelle Server nach Maß – sicher, flexibel und hochverfügbar

  • Hochverfügbare, leistungsstarke, virtuelle Windows und Linux Server
  • Flexibel anpassbar mit monatlicher Abrechnung
  • Virtuelle Firewalls mit anpassbarem Funktionsumfang
  • Virtuelle Netzwerke für komplexere Strukturen
  • Verfügbarkeit vorab im Partner-Rechenzentrum (April 2020)

GREENCLOUD Backup

Unkomplizierte Datensicherung für Server und Arbeitsplätze

  • Datensicherung für Server, PCs und Notebooks über Veeam Cloud Connect
  • Verschlüsselte Übertragung und Speicherung der Daten
  • Sicherung der Daten in unserem Rechenzentrum auf zwei physisch getrennten Speichern
  • Optional zusätzliche georedundante oder lokale Sicherung
  • Optional für Microsoft Office 356
  • Verfügbarkeit vorab im Partner-Rechenzentrum (April 2020)

Individuelle Lösungen

Wir analysieren Ihren Bedarf und konzipieren nach Maß – zum Beispiel:

  • Mail-Archiv: E-Mail Archivierung nach rechtlichen Vorgaben und EU-DSGVO
  • Exchange: E-Mail Server mit vollem Funktionsumfang von Microsoft Exchange
  • Phone: Vollwertige Telefonanlage in der Cloud
  • Drive: Sicherer Datenspeicher. Zugriff über Windows, MacOS, Android, iOS
  • Hosting & Housing: Sicherer Betrieb von eigenen oder gemieteten Hardware-Severn
  • Software & Lizenzen:, Microsoft SPLA und CSP Lizenzen, (z.B. Office 365, Teams, etc.)
